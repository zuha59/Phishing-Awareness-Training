# Phishing Awareness Training
This project aims to educate users about phishing attacks,
social engineering techniques, and cybersecurity best practices.

The training material helps users identify phishing emails,
fake websites, and common tactics used by cybercriminals.
## Objectives

- Understand phishing attacks
- Identify phishing emails
- Recognize fake websites
- Learn social engineering tactics
- Apply cybersecurity best practices
- Test knowledge using an awareness quiz
## Topics Covered

1. Introduction to Phishing
2. Types of Phishing Attacks
3. Email Phishing
4. Spear Phishing
5. Smishing
6. Vishing
7. Fake Website Detection
8. Social Engineering

## Signs of a Phishing Email

- Suspicious sender address
- Urgent language
- Unexpected attachments
- Requests for passwords
- Grammar and spelling mistakes
- Suspicious links
## Signs of a Fake Website

- Misspelled domain names
- Missing HTTPS
- Poor website design
- Unusual URLs
- Requests for sensitive information
## Social Engineering Techniques

### Authority
Pretending to be a manager or government official.

### Fear
Threatening account suspension.

### Urgency
Creating panic to force quick action.

### Curiosity
Encouraging users to open malicious links.

### Reward
Offering fake prizes or rewards.
## Best Practices

- Verify sender identities
- Avoid clicking unknown links
- Use Multi-Factor Authentication (MFA)
- Keep software updated
- Report suspicious emails
- Use strong passwords
## Awareness Quiz

Q1. An email asks for your password immediately.
Answer: Phishing Attempt

Q2. Which URL is legitimate?

A. https://www.paypa1.com
B. https://www.paypal.com

Answer: B

Q3. Which is a sign of phishing?

A. Grammar mistakes
B. Urgent messages
C. Suspicious links
D. All of the above

Answer: D

After completing this training, users will be able to:

- Identify phishing attempts
- Detect fake websites
- Understand social engineering attacks
- Follow cybersecurity best practices
- Reduce the risk of credential theft
